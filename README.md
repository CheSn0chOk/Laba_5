# Система управления кассой вокзала

Десктопное приложение для автоматизации работы железнодорожной кассы с использованием Java Swing и базы данных SQLite.

## О проекте

Приложение разработано в рамках лабораторной работы по дисциплине "Практикум по программированию". Система позволяет управлять тарифами, регистрировать пассажиров, продавать билеты и формировать отчеты.

## Технологии

- **Java** - основной язык программирования
- **Swing** - графический интерфейс
- **SQLite** - реляционная база данных
- **JDBC** - подключение к БД
- **Singleton Pattern** - шаблон проектирования

## Архитектура

### Основные классы:

- **Main** - главное окно приложения, графический интерфейс
- **TicketOffice** - бизнес-логика (Singleton)
- **DatabaseManager** - работа с БД (Singleton) 
- **Passenger** - данные пассажира
- **Ticket** - данные билета
- **Direction** - перечисление направлений
